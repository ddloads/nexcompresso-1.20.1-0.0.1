NexCompresso Mod for Minecraft
==============================

About
-----
NexCompresso is a Minecraft mod inspired by Ex Compressum, tailored for Minecraft 1.20 and designed to work with Ex Deorum. It introduces a variety of compressed blocks and tools to enhance the resource gathering experience in a skyblock-style or survival playthrough.

Features
--------
- Compressed Blocks: Introducing compressed versions of common blocks, increasing efficiency in storage and handling.

Installation
------------
1. Ensure you have Minecraft Forge 47.1.3 or newer installed.
2. Download the latest release of NexCompresso from the [Releases](https://github.com/ddloads/NexCompresso/releases) page.
3. Place the downloaded `.jar` file into the `mods` directory of your Minecraft installation.
4. Launch Minecraft and enjoy!

Compatibility
-------------
Tested and compatible with:
- Ex Deorum
- Just Enough Items (JEI)
- Jade
